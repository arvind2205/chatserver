To run the chatserver, just type python server.py in terminal or git bash. Once you have done this, your server is now listening on port 1234 - connect to this port from any other device. The supported commands are iam and msg. iam: Arvind means that your user will be renamed to Arvind, and msg: Hello will print the message hello to the screen. Cheers - Don't forget to visit the author's webpage whose tutorial this exercise is based off of.
